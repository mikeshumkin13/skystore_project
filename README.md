# Домашнее задание 27: Аутентификация в Django

В рамках задания реализована система аутентификации пользователей в проекте Skystore:

## ✅ Основной функционал:
- Кастомная модель пользователя (`users.User`)
- Регистрация через `UserRegisterView`
- Авторизация через `LoginView`
- Выход через `LogoutView`
- Ограничение доступа к управлению продуктами (добавление, редактирование, удаление) только для авторизованных пользователей
- Отправка приветственного письма после регистрации через SMTP Яндекс

## Домашнее задание 28: Права доступа

В этом задании реализованы механизмы разграничения прав доступа для управления продуктами:

- Добавлены поля `is_published` и `owner` к модели `Product`.
- При создании продукта автоматически сохраняется авторизованный пользователь как владелец.
- Реализованы проверки доступа:
  - Редактировать и удалять продукт может только владелец.
  - Модератор может удалять продукт даже без владения.
- Добавлена группа **"Модератор продуктов"** с правами:
  - удаление любого продукта;
  - отмена публикации (`can_unpublish_product`).
- Реализована команда `create_groups` для создания и настройки группы модераторов.
- Кнопки "Редактировать" и "Удалить" на странице продукта отображаются только для владельца или модератора.
